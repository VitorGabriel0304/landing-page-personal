# ⚡ Elite Personal Trainer Landing Page

Landing page de alta conversão para Personal Trainers que buscam um posicionamento premium.

## 🎨 Design System

- **Estética:** Premium Light (Marfim, Esmeralda e Champanhe).
- **Tipografia:** Inter (Moderna) & Playfair Display (Sofisticada).
- **Tecnologias:** HTML5 puro, CSS3 (Grid/Flex), JavaScript Vanilla, AOS Library.

## 🛠️ Como usar

1. Clone este repositório.
2. Substitua a imagem no `style.css` (background do `.hero`) por uma foto sua em alta resolução.
3. Altere o nome e os textos no `index.html`.
4. Faça o deploy em plataformas como Vercel ou Netlify gratuitamente.

## 📁 Estrutura

- `/assets/img`: Suas fotos e ícones.
- `/assets/css`: Estilização e Variáveis.
- `/assets/js`: Comportamento e Animações.
